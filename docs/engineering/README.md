# Engineering Baseline Documents

本目录包含工程执行与落地层文档，将核心协议和系统组织文档转化为可执行的工程规则。

## 文档列表

- **engineering_baseline.md** - 工程基线（目录结构、测试要求、类型检查、日志要求、开发顺序、里程碑、AI 协作约束）
- **m1_checklist.md** - M1 基线收敛的执行检查表
- **delivery_master_plan.md** - 从 M1 到产品落地的总交付计划、阶段门禁与回滚条件
- **implementation_task_packages.md** - 面向任务 AI / 开发者的阶段任务包拆解

## 层级归属

这些文档属于第三层：工程执行与落地层。它们决定仓库怎么搭、测试怎么补、类型怎么检查、交付怎么验收，并提供参考实现。
