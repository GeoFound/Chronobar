# Examples

本目录用于存放 Chronobar 的最小可运行样例、配置样例和协议契约样例。

当前阶段的正式口径仍为 M1 基线收敛，但本目录允许存在**探索性或预演性质**的运行时代码样例，用于帮助后续阶段验证方向。
这些样例不等同于 P1 / M2 已正式验收，当前阶段真相源仍以 `../docs/engineering/current_phase_and_truth_source.md` 为准。

后续将优先按以下顺序补充或收敛：

1. 最小配置组合样例
2. 最小插件 manifest 样例
3. 最小策略样例
4. 最小事件日志样例
5. 最小回放输入样例

当前已存在的示例应被视为参考实现或预演产物，而不是正式 gate 通过证明。
